My First Blog Post

Posted on March 18, 2024

Here's an example of how code blocks will look in our blog posts. Let's start with some Python code: 

def hello_world():
    print("Hello, World!")
    
# Example of a class
class BlogPost:
    def __init__(self, title, content):
        self.title = title
        self.content = content
        
    def publish(self):
        print(f"Publishing: {self.title}")

You can also use inline code like this: print("Hello") within your paragraphs.

Here's an example of JavaScript code: 

function createBlogPost(title, content) {
    return {
        title,
        content,
        date: new Date(),
        publish() {
            console.log(`Publishing: ${this.title}`);
        }
    };
}
